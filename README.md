# APP/WEBSITE BOILERPLATE

My own boilerplate to start working on new app/website.

How to use:
1) Edit package.json info
2) Edit index.html meta tags
3) Run command line
4) Run: npm install
5) Run: gulp
6) Enjoy coding!

This boilerplate use:
- GULP.JS
- SASS
- BABEL
- BROWSER SYNC
- AUTOPREFIXER
- UGLIFY
- NORMALIZE CSS
